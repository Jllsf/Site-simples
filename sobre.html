Sobre

Este site foi desenvolvido como parte de um trabalho acadêmico com o objetivo de aplicar, na prática, os conhecimentos adquiridos na disciplina de Front-End.

Durante o desenvolvimento, foram utilizadas as tecnologias fundamentais da web: HTML5 para a estrutura do conteúdo, CSS3 para o estilo e layout visual, e JavaScript para adicionar interatividade às páginas.

projeto tem como propósito reforçar o aprendizado dos conceitos básicos de desenvolvimento de interfaces web, organização de código e boas práticas de programação.
