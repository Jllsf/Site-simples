
# Melhor Sabor Que Há 🍽️

Um site simples e intuitivo desenvolvido como parte de um projeto acadêmico, com o objetivo de compartilhar **receitas práticas e saborosas** para o dia a dia. O site foi criado aplicando os conhecimentos fundamentais de desenvolvimento web com **HTML5**, **CSS3** e **JavaScript**.

---

## 📌 Funcionalidades

- ✅ Página inicial com apresentação do site  
- ✅ Página de receitas com imagens, ingredientes e modo de preparo  
- ✅ Interação com `details/summary` para visualização suave das receitas  
- ✅ Página de contato com formulário validado via JavaScript  
- ✅ Página "Sobre" explicando o propósito e tecnologias do projeto  
- ✅ Menu de navegação entre todas as páginas  
- ✅ Estilização agradável e responsiva  

---

## 🧪 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript Vanilla

## ▶️ Como Executar Localmente

1. **Clone o repositório**:
   ```bash
   git clone https://github.com/Jllsf/Site-simples.git
   ```
2. **Acesse a pasta**:
   ```bash
   cd Site-simples
   ```
3. **Abra o arquivo `index.html` no navegador**:
   - Clique duas vezes no arquivo `index.html`  
   - Ou use uma extensão como "Live Server" no VS Code
